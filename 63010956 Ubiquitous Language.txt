- Healthcare
  1. Long-term care => บริการการดูเเลทั้งทางการเเพทย์ เเละไม่ใช่ทางการเเพทย์ สำหรับคนที่ไม่สามารถช่วยเหลือตนเองในชีวิตประจำวันได้
เช่น การอาบน้ำ การดูแลระยะยาวทำในบ้าน ในชุมชน หรือในสถานพยาบาล คนทุกคนอาจจะต้องใช้ เเละประกันสุขภาพส่วนใหญ่ไม่จ่ายค่าดูแลระยะยาว
  2. Health insurance => สัญญาที่กำหนดให้บริษัทประกันสุขภาพจ่ายค่ารักษาพยาบาลบางส่วนหรือทั้งหมดเพื่อแลกกับเบี้ยประกันภัย
  3. Health status => เงื่อนไขทางการแพทย์ (ทั้งสุขภาพร่างกายและจิตใจ) การรับการรักษาพยาบาล ประวัติทางการแพทย์ 
ข้อมูลทางพันธุกรรม เเละหลักฐานการประกัน
  4. Full-time employee => พนักงานคนใดก็ตามที่ทำงานโดยเฉลี่ยอย่างน้อย 30 ชั่วโมงต่อสัปดาห์เป็นเวลามากกว่า 120 วันในหนึ่งปี
  5. Plan => ผลประโยชน์ที่นายจ้าง สหภาพแรงงาน หรือผู้สนับสนุนกลุ่มอื่นๆ มอบให้เพื่อชำระค่าบริการดูแลสุขภาพ
  6. Health coverage => สิทธิ์ทางกฎหมายในการชำระเงิน หรือการชำระเงินคืนสำหรับค่ารักษาพยาบาล 
โดยทั่วไปภายใต้สัญญากับบริษัทประกันสุขภาพ แผนสุขภาพแบบกลุ่มที่เสนอเกี่ยวกับการจ้างงาน หรือโครงการของรัฐบาล เช่น Medicare, Medicaid
  7. Generic drugs => ยาที่ต้องสั่งโดยแพทย์ซึ่งมีสูตรส่วนผสมออกฤทธิ์เดียวกับยาแบรนด์เนม มักจะมีราคาต่ำกว่ายาแบรนด์เนม 
สำนักงานคณะกรรมการอาหารและยา (อย.) ให้คะแนนยาเหล่านี้ว่าปลอดภัยและมีประสิทธิภาพเทียบเท่ายาแบรนด์เนม
  8. Dividend => การจ่ายเงินโดยบริษัทที่แสวงหาผลกำไรให้กับผู้ถือหุ้น การชำระเงินนี้เป็นส่วนหนึ่งของรายได้ของบริษัท และอาจจ่ายเป็นจำนวนครั้งต่อปี
  9. Dental coverage => ผลประโยชน์ที่ช่วยชำระค่าใช้จ่ายในการไปพบทันตแพทย์สำหรับบริการขั้นพื้นฐาน หรือบริการป้องกัน 
เช่น การทำความสะอาดฟัน เอ็กซ์เรย์ และอุดฟัน
  10. Allowed amount => จำนวนเงินสูงสุดที่แผนจะจ่ายสำหรับบริการดูแลสุขภาพที่ครอบคลุม อาจเรียกอีกอย่างว่า "ค่าใช้จ่ายที่มีสิทธิ์" "ค่าเผื่อการชำระเงิน" หรือ "อัตราต่อรอง"

- Education
  1. Analysis => การกระทำเกี่ยวกับการเเยกส่วนเพื่อศึกษา
  2. Bias in education => อคติที่รับรู้ในระบบการศึกษา
  3. Blogish => การสื่อสารแบบโต้ตอบ ซึ่งตรงข้ามกับการบรรยายแบบดั้งเดิม
  4. Bully => บุคคลที่คาดว่ามีความผิดปกติทางอารมณ์ ซึ่งทรมานผู้อื่นผ่านการล่วงละเมิดทางวาจา การทำร้ายร่างกายหรือวิธีการบีบบังคับ อื่นๆ
  5. Public school => คำนี้มีความหมายที่แตกต่างกัน (และในบางกรณีที่ขัดแย้งกัน) เนื่องจากความแตกต่างในระดับภูมิภาค
  6. Problem solving => เป็นส่วนหนึ่งของการคิด เกิดขึ้นหากสิ่งมีชีวิต หรือปัญญาประดิษฐ์ไม่ทราบวิธีดำเนินการจากสถานการณ์ที่กำหนดไปยังสถานการณ์เป้าหมายที่ต้องการ 
เป็นส่วนหนึ่งของกระบวนการปัญหาที่ใหญ่กว่า ซึ่งรวมถึงการค้นหาปัญหา และการกำหนดปัญหา
  7. Problem finding => การค้นหาปัญหาเป็นส่วนหนึ่งของกระบวนการปัญหาที่ใหญ่กว่า ซึ่งรวมถึงการสร้างปัญหา และการแก้ปัญหา 
การค้นหาปัญหาต้องใช้วิสัยทัศน์ทางปัญญา และความเข้าใจในสิ่งที่ขาดหายไป
  8. School => เป็นสถานที่สำหรับการเรียนรู้ ช่วงของสถาบันที่ครอบคลุมโดยคำนั้นแตกต่างกันไปในแต่ละประเทศ
  9. School bus => รถประจำทางที่ใช้รับส่งเด็กและวัยรุ่น ไป และกลับจากโรงเรียน รถโรงเรียนหลายประเภทได้แพร่หลายและใช้เครื่องยนต์ และถูกใช้ในทุกส่วนของโลก
  10. Special education => อธิบายทางเลือกทางการศึกษาที่เน้นการสอนของนักเรียนที่มีความต้องการทางวิชาการ พฤติกรรม สุขภาพ หรือร่างกายที่ไม่สามารถทำได้มากพอ
โดยใช้โปรแกรม หรือเทคนิคการศึกษาแบบดั้งเดิม

- Banking
  1. Acceleration => ทวงชำระหนี้ที่ยังไม่ครบกำหนดชำระเต็มจำนวน มักจะเป็นการเยียวยาที่ให้ไว้ในเอกสารเงินกู้สำหรับผู้ให้กู้เพื่อใช้ในกรณีที่ผู้ยืมผิดนัด
  2. Acceptance => เวลาที่ร่างไว้ที่ได้รับการยอมรับสำหรับการชำระเงิน ดูการยอมรับของธนาคาร
  3. Accreting swap => การแลกเปลี่ยนอัตราดอกเบี้ยกับจำนวนเงินตามสัญญาที่เพิ่มขึ้น
  4. Online bank => ธนาคารออนไลน์ หรือที่เรียกว่าธนาคารดิจิทัลหรืออินเทอร์เน็ต ดำเนินการผ่านอินเทอร์เน็ตเป็นหลัก 
สามารถจัดการบัญชีของคุณที่ธนาคารออนไลน์จากคอมพิวเตอร์ หรืออุปกรณ์มือถือได้จากทุกที่ทุกเวลา
  5. Wire transfers => การโอนเงินทางอิเล็กทรอนิกส์อีกประเภทหนึ่ง (ETF) 
การโอนเงินผ่านธนาคารมักเกี่ยวข้องกับการชำระค่าธรรมเนียม ขึ้นอยู่กับว่าการโอนเงินเข้าหรือออกและในประเทศหรือต่างประเทศ
  6. Conditions => หรือที่เรียกว่าข้อกำหนดและเงื่อนไขนี่คือการพิมพ์บัญชีธนาคารหรือสัญญาเงินกู้ 
ตรวจสอบให้แน่ใจว่าคุณได้อ่านและเข้าใจความหมายของบัญชีการเงินและภาระผูกพันของคุณ
  7. Check => ประเภทของเครื่องมือทางการเงินที่สั่งให้ธนาคารผู้ทำเช็คชำระเงินให้กับผู้รับที่ระบุไว้บนเช็ค บางคนเขียนเช็คกระดาษและคนอื่น ๆ 
ใช้คุณสมบัติการชำระบิลออนไลน์ของธนาคารเพื่อออกเช็คที่สร้างขึ้นทางอิเล็กทรอนิกส์
  8. Checking account => บัญชีเช็คของคุณ ซึ่งบางครั้งเรียกว่าบัญชีธนาคาร เป็นฐานหลักสำหรับการทำธุรกรรมทางการเงิน 
บัญชีสามารถมีดอกเบี้ย หรือไม่มีดอกเบี้ยขึ้นอยู่กับธนาคารหรือเครดิตยูเนี่ยน
  9. Solvency => เมื่อธนาคารมีเงินเพียงพอที่จะครอบคลุมความเสียหายที่อาจเกิดขึ้น คาดว่าธนาคารจะรักษาระดับเงินทุนให้เพียงพอเพื่อคงสภาพเป็นตัวทำละลายและหลีกเลี่ยงความล้มเหลว FDIC 
และหน่วยงานกำกับดูแลของรัฐบาลกลางอื่น ๆ ทำงานร่วมกับธนาคารเพื่อรักษามาตรฐานการชำระหนี้
  10. Joint account => บัญชีที่มีเจ้าของตั้งแต่สองคนขึ้นไปที่เป็นเจ้าของบัญชีเท่ากัน โดยมีสิทธิและภาระหน้าที่เหมือนกันในการใช้บัญชี 
ตัวอย่างเช่น คู่สมรสจำนวนมากมีบัญชีตรวจสอบร่วมที่อนุญาตให้ทั้งคู่เขียนเช็คและฝากเงินเข้าในบัญชีเดียวกันได้
